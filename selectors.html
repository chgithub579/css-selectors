CSS Selectors

Adding Css Selectors To Different HTML Elements

1.Element Selector are used to select the html elements based on the html element name

  1. Class Selector are used to select the html elements with specific class attribute
  2. Id Selectors are used to select specific element based on the id attribute of the html element
  3. Grouping selectors are used to group html elements to apply same style definitions
  4. Universal Selectors are used to select all html elemnts of page to apply same style
